Unofficial Linux helper

Install Google Antigravity 2.0 on Debian/Ubuntu with one command.

This installer resolves Google’s official tarball at install time, places it under /opt, adds app launchers, and gives you a clean update command.

Get the command View installer
Install
INSTALLER_URL="https://opensnap.github.io/antigravity/install.sh"
curl -fsSL "$INSTALLER_URL" | sudo -E env ANTIGRAVITY_EASY_INSTALLER_URL="$INSTALLER_URL" bash -s -- --all

Official tarball flow

No binary mirror. No unofficial repackaging. The script resolves Google’s current Antigravity download URL when it runs.

Desktop integration

App menu entries, icons, command launchers, MIME integration, and optional Nautilus right-click folder opening.

Updates included

Run sudo antigravity-easy update --all later to fetch the latest official tarballs again.

Update

Keep Antigravity current

The installer stores its published URL, so installed machines can update without remembering the full curl command.

Commands
sudo antigravity-easy update --all
sudo update-antigravity
sudo update-antigravity-ide
